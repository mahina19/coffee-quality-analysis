## 📋 Overview

This project analyzes the Coffee Quality Institute (CQI) dataset to uncover what makes high-quality coffee. By examining 207 arabica coffee samples across multiple variables, this analysis provides actionable insights into coffee production, processing methods, and flavor profiles.

**Perfect for:**
- Coffee enthusiasts wanting to understand quality metrics
- Data analysts looking for a portfolio example
- Businesses in the coffee industry seeking data-driven insights

---

## 🎯 Key Questions Answered

1. **Which country produces the highest quality coffee?**
2. **How does processing method affect coffee quality?**
3. **Which flavor attributes best predict overall quality?**

---

## 🔍 Key Findings

✨ **Ethiopia** produces the highest quality coffee globally (average score: 82.5/100)

🔄 **Double Anaerobic Washed** is the best processing method, resulting in superior quality scores

👃 **Flavor & Aftertaste** are the strongest predictors of overall coffee quality (correlation > 0.85)

⭐ **Acidity** and **Body** also significantly influence the final quality rating

## 🚀 Getting Started

### Prerequisites

- Python 3.8 or higher
- pip (Python package manager)

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/mahina19/coffee-quality-analysis.git
cd coffee-quality-analysis
```

2. **Install dependencies**
```bash
pip install -r requirements.txt
```

Or install manually:
```bash
pip install pandas matplotlib seaborn jupyter
```

3. **Open the notebook**
```bash
jupyter notebook "Coffee Analysis.ipynb"
```

4. **Run the analysis**
- The notebook guides you through each step
- All visualizations generate automatically
- Findings are clearly marked with key insights
