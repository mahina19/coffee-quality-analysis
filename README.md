# coffee-quality-analysis
Analysis of Coffee Quality Institute dataset using Python, Pandas and Matplotlib
# ☕ Coffee Quality Analysis

## Overview
Analysis of 207 coffee samples from the Coffee Quality 
Institute (CQI) dataset to understand what makes 
a high quality coffee.

## Questions Answered
1. Which country produces the highest quality coffee?
2. Does processing method affect coffee quality?
3. Which flavor attributes most predict overall quality?

## Key Findings
- Ethiopia produces the highest quality coffee globally
- Double Anaerobic Washed is the best processing method
- Flavor and Aftertaste are the strongest quality predictors

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- VS Code

## Dataset
Coffee Quality Institute via Kaggle
